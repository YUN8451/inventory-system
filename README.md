# 库存管理系统

基于纯前端（HTML/CSS/JS + LocalStorage）的智能库存查询与管理平台，支持多仓库、多维度库存筛选与维护。

## 功能特性

### 前台查询页（index.html）
- 多维筛选：产品组、仓库编码/名称、产品线、物料编码/名称/型号等
- 在库库存量（橙色）、锁库量（红色）、可用库存展示
- 顶部实时汇总统计
- Excel 导出功能

### 后台维护页（admin.html）
- **库存维护**：新增 / 编辑 / 删除库存记录
- **批量导入**：Excel 文件上传，支持追加更新和清空重建两种模式
- **流水变动明细**：自动记录所有增删改操作，支持筛选与导出
- 导入模板下载

### 数据层（data.js）
- LocalStorage 持久化存储
- 前后台数据自动同步
- 内置种子数据（杭州/济南/佛山三仓示例）

## 技术栈
- HTML5 + CSS3 + JavaScript (ES6+)
- XLSX.js (Excel 导入导出)
- 无后端依赖，浏览器直接运行

## 使用方式
1. 克隆仓库或下载文件
2. 确保三个文件在同一目录下
3. 双击 `index.html` 打开前台查询页
4. 双击 `admin.html` 打开后台维护页

## 部署
可直接部署至 GitHub Pages 或任何静态文件托管服务。
